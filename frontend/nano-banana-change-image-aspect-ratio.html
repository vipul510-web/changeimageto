Common Aspect Ratios for Different Platforms

  • Square (1:1): Instagram posts, profile pictures - 1080 x 1080 pixels
  • Portrait (9:16): Instagram stories, TikTok, Snapchat - 1080 x 1920 pixels
  • Landscape (16:9): YouTube videos, widescreen displays - 1920 x 1080 pixels
  • Classic (4:3): Traditional photos, presentations - 1024 x 768 pixels
  • Widescreen (21:9): Ultra-wide displays, cinematic content
  • Facebook Cover (2.63:1): Facebook cover photos - 1640 x 624 pixels
