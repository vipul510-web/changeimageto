    imageguard — the Python library powering this tool

    Want to run these same quality checks — blur, noise, resolution, compression — inside your own AI pipeline? imageguard is the open-source Python package extracted from this tool. One call, one result object. No OpenCV knowledge required.

    from imageguard import validate

result = validate("photo.jpg")

if not result.ok:
    print("Reject:", result.reason)   # e.g. "blurry"
else:
    print("Score:", result.score)     # 0.0 – 1.0

    • Blur Detection: Identifies out-of-focus or motion-blurred images using Laplacian variance analysis
