This guide shows the exact steps to remove a picture background in Google Slides without third‑party add‑ons. You can mask, crop, and make backgrounds transparent using built‑in tools and a quick detour via Google Drawings.

Method 1 — Make white background transparent

  1. Insert the image: Insert → Image → Upload from computer.
  2. Select the image → Format options (right panel) → Adjustments.
  3. Increase Transparency until the white/solid background fades. Fine‑tune Brightness/Contrast to keep the subject visible.
  4. Optional: Add a colored shape behind the image (Insert → Shape) for a new background.

Works best when the background is plain white or a single color.

Method 2 — Background removal via Google Drawings (free)

  1. Right‑click the image → Open with → Google Drawings.
  2. In Drawings, use the Format → Format options → Adjustments slider and the Image → Recolor if needed to isolate the subject.
  3. For tighter edges, use the Crop tool with Mask (dropdown next to the crop icon) and choose a shape that matches your subject. Drag the control handles to refine.
  4. File → Download → PNG. This preserves transparency.
  5. Back in Slides, Insert → Image → Upload → choose the PNG you downloaded.

Tip: PNG is required to keep transparent backgrounds.

Tips for Slides

  • Use high‑resolution images (at least 1600×1200) to avoid pixelation on projectors.
  • Add a soft shadow (Format options → Drop shadow) to make the cutout blend naturally.
  • Lock background elements by setting them as the slide background image.
