    Social Media Image Size Requirements

    Each social media platform has specific image dimension requirements. Here are the most common sizes:

    • Instagram Posts: 1080×1080px (square), 1080×1350px (portrait), 1080×566px (landscape)
    • Instagram Stories: 1080×1920px (9:16 aspect ratio)
    • Instagram Reels: 1080×1920px (9:16 aspect ratio)
    • Facebook Posts: 1200×630px (1.91:1 aspect ratio)
    • Facebook Cover Photo: 1640×859px (1.91:1 aspect ratio)
    • Facebook Profile Picture: 400×400px (square)
    • Twitter Posts: 1200×675px (16:9 aspect ratio)
    • LinkedIn Posts: 1200×627px (1.91:1 aspect ratio)
    • Pinterest Pins: 1000×1500px (2:3 aspect ratio)
    • TikTok Videos: 1080×1920px (9:16 aspect ratio)

    Social Media Image Optimization Tips

    • Use Platform-Specific Sizes: Each platform has optimal dimensions - use them for best results
    • Maintain Quality: Use 85-95% quality to ensure images look sharp on all devices
    • Square Format for Instagram: Square images (1:1) work best for Instagram feed posts
    • Vertical for Stories: Use 9:16 aspect ratio (1080×1920px) for Instagram Stories and Reels
    • Test on Mobile: Always check how images look on mobile devices before posting
    • Batch Process by Platform: Process images separately for each platform to ensure perfect dimensions

    Platform-Specific Recommendations
