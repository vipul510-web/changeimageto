How to Install

Method 1: Chrome Web Store (Recommended)

  1. Click the Install Chrome Extension button above, or visit the Chrome Web Store page
  2. Click "Add to Chrome" on the extension page
  3. Confirm the installation in the popup dialog
  4. The extension icon will appear in your Chrome toolbar
  5. You're ready to start editing images!

Method 2: Developer Mode (For Testing)

  1. Open Google Chrome and navigate to chrome://extensions/
  2. Enable "Developer mode" using the toggle in the top right corner
  3. Click "Load unpacked" button
  4. Select the folder containing the extension files
  5. The extension will be installed and ready to use

How to Use

Method 1: Right-Click Context Menu

  1. Navigate to any website with images
  2. Right-click on any image you want to edit
  3. Select "Edit with ChangeImageTo" from the context menu
  4. Choose your desired operation (Remove Background, Blur Background, etc.)
  5. The image will be processed and displayed on the page
  6. Download the processed image directly

Method 2: Extension Popup

  1. Click on the ChangeImageTo extension icon in your Chrome toolbar
  2. Click "Pick Images From Page" to select images on the current webpage
  3. Click on images to select them (they'll be highlighted)
  4. Use the quick action buttons in the popup to process selected images
  5. Download processed images individually or in bulk

Method 3: Upload Images

  1. Click the extension icon in your toolbar
  2. Click "Upload Image" button
  3. Select an image file from your computer
  4. Choose your editing operation from the available options
  5. Download the processed result

Frequently Asked Questions

Is the extension free?

Yes! The ChangeImageTo Chrome extension is completely free to use. All features are available at no cost.

Do I need to create an account?

No account is required. You can start using the extension immediately after installation.

Are my images stored?

No. Images are processed in real-time and are not stored on our servers. Your privacy is protected.

Does it work offline?

The extension requires an active internet connection as image processing happens on our servers.

Can I use it on other browsers?

Currently, the extension is available for Google Chrome. Support for other Chromium-based browsers (Edge, Brave, etc.) may work but is not officially supported.

What if the extension doesn't work?

Make sure you have the latest version of Chrome installed. If issues persist, try refreshing the webpage or reinstalling the extension. Check that you have an active internet connection.
